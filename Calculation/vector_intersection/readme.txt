Оригинальная задача:
Есть квадрат 7 x 7 на клетчатой решетке, можно поставить точку в центре любой его клетки 1 x 1.
Сколько максимум точек можно поставить, чтобы никакие 4 не образовывали прямоугольник со сторонами, параллельными сторонам квадрата?

Альтернативная формулировка:
Найти максимальное суммарное число бит в семи 7-битных числах таких, что побитовое И любых двух из них равно нулю или степени двойки.
Иными словами для чисел a1 ... a7: для любых (i, j) верно (a[i] & a[j]) & ((a[i] & a[j]) - 1) = 0.

Пример для 21 точки:
0000111
0011001
0110100
1100001
1010010
0101010
1001100

24 точки разместить невозможно, т.к. найдутся три числа хотя бы с 4мя единицами.
1 число: едитницы в битах a, b, c, d
2 число: единицы в битах d, e, f, g
3 число: обязательно совпадает с 1м или 2м хотя бы в двух битах

Идеи решения на компьютере:

Попробуем найти пример для 22 точек. Переберем разбиения числа 22 по строкам.
22 = a1 + a2 + ... + a7, причем a1 >= a2 >= ... >= a7
Затем сгенерируем рекурсивно все комбинации чисел в строках, причем для каждого числа будем генерировать все состояния используя Gosper's Hack.

Для 21 ответ находится за 10 сек.
Для 22 ответ не найден в течении 20 минут.
